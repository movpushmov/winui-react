# Welcome to WinUI React repository!

This is React UI library, which implements Windows UI Design (WinUI 2.7 Design, Windows 11 Design).

### Live demo:

https://movpushmov.github.io/

- NavigationView (top variant), BreadcrumbBar
- Teaching Tip
- RichEditBox
